Features
========

* Обзор показывает маршрут и количество достопримечательностей вдоль маршрута. Путь максимально соответствует первоначальному маршруту культурной тропы.
* На карте отображаются детали вдоль тропы. Достопримечательности обозначены красными маркерами с номерами.
* Модальный диалог показывает информацию о достопримечательностях. Тексты взяты из буклетов, выпущенных для культурных троп. Фото иллюстрирует запись.
* Маршрут рассчитывается с помощью системы маршрутизации, разработанной для этого проекта. Это позволяет отображать длину маршрута и необходимое время.
* Информация в диалоге о достопримечательностях содержит ссылки на источник. Дополнительная информация связана.
* Каждой культурной тропе предшествует текст, подчеркивающий особенности маршрута. Эти тексты находятся в меню «О нас».
* Можно отображать различные языки. Собственные имена не переводятся. Каждый дополнительный язык требует дополнительных редакционных усилий.
* Инструмент Locate позволяет передавать местоположение в браузер. Текущая позиция отображается на карте синей точкой.
* При загрузке страницы все данные загружаются в браузер. Обработка происходит на смартфоне. Данные о людях, их поведении или местоположении не собираются и не хранятся.
* У каждой культурной тропы есть своя страница. Тропы района можно найти в меню «О нас / Ссылки».
* Страница «О проекте» описывает контекст. Здесь можно связаться с командой разработчиков.
