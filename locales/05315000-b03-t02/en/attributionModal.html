Using OpenStreetMap and Bootleaf

OpenStreetMap

OpenStreetMap (OSM) is a collaborative, open source project with the goal of creating a detailed and freely available world map. Through the active participation of the global mapper community, OSM offers flexible and extensive geographical data that can be used in a variety of applications.

Bootleaf

Bootleaf is a simple, responsive template for building web mapping applications with Bootstrap, Leaflet, and typeahead.js developed by Bryan McBride
